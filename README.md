# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze how Bitcoin market sentiment affects trader performance.

## Approach
- Merged sentiment dataset with trading dataset
- Created profit_flag to calculate win rate
- Performed analysis on PnL, win rate, trade count, and trade size
- Visualized trends using boxplots and countplots

## Key Insights
- Extreme Greed -> highest profitability and win rate
- Extreme Fear -> lowest performance
- Fear -> highest trading activity
- Larger trade sizes during Fear indicate higher risk-taking
- Controlled trading during Greed leads to better outcomes

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
